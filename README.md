# Advanced Tic Tac Toe Web Application

## Overview
Advanced Tic Tac Toe is a responsive web-based game developed using HTML, CSS, and JavaScript. It allows two players to play Tic Tac Toe with features like winner detection, score tracking, game reset, and a user-friendly interface.

## Features

- Two-player gameplay (computer vs player)
- Winner and draw detection
- Score tracking
- Reset and New Game options
- Responsive design
- Interactive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.

- Online multiplayer
